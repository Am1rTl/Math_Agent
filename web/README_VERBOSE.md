# Verbose Mode для Math Agent

## Описание

Режим `--verbose` позволяет выводить в консоль все системные промпты, пользовательские промпты и ответы от AI в реальном времени.

## Использование

### Запуск с verbose режимом:

```bash
python app.py --verbose
```

### Комбинация с debug режимом:

```bash
python app.py -d --verbose
```

- `-d` - сохраняет все промпты и ответы в файл `math_agent.log`
- `--verbose` - выводит все промпты и ответы в консоль в реальном времени

## Формат вывода

При включенном verbose режиме вы увидите:

```
================================================================================
🤖 AI CALL - Model: google/gemini-2.5-pro
================================================================================

📋 SYSTEM PROMPT:
--------------------------------------------------------------------------------
Ты - семантический парсер. Твой ответ ДОЛЖЕН быть *ТОЛЬКО* JSON-объектом...
--------------------------------------------------------------------------------

👤 USER PROMPT:
--------------------------------------------------------------------------------
ЗАДАЧА: Найдите все значения параметра a...
--------------------------------------------------------------------------------

🔮 AI RESPONSE:
--------------------------------------------------------------------------------
{
  "id": "task_123",
  "statement": "...",
  ...
}
--------------------------------------------------------------------------------
```

## Когда использовать

- **Отладка промптов**: когда нужно понять, какие именно инструкции получает AI
- **Анализ ответов**: когда нужно увидеть сырые ответы от модели
- **Разработка**: при разработке новых промптов или изменении существующих
- **Обучение**: для понимания работы мультиагентной системы

## Примечания

- Verbose режим значительно увеличивает объем вывода в консоль
- Рекомендуется использовать только при разработке и отладке
- Для production используйте обычный режим без флагов
